# Fyr operations dashboard

This is a complete Fyr example using reactive state, computed metrics, two-way form binding, filtering, reactive lists, attribute binding, notifications, browser persistence, and JSON export.

Run it from the repository root:

~~~bash
npx vite --host 127.0.0.1
~~~

Then open:

~~~
http://127.0.0.1:5173/examples/advanced-dashboard/
~~~

The app loads `@aldane-dev-create/fyr@0.1.2` through jsDelivr first. Until that patch release is published and jsDelivr finishes propagating it, it falls back to the local `dist` build so contributors can run it immediately.
